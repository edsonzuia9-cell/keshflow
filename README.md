KeshFlow

Aplicação web de gestão financeira pessoal para Moçambique (M-Pesa, m-Kesh, e-Mola, contas bancárias).

Aceder à Aplicação
URL de produção: https://keshflow.ac.mz

Funcionalidades
• Gerenciar carteiras digitais e contas bancárias
• Registar entradas e saídas com categorização
• Transferir valores entre contas
• Definir alertas de saldo baixo
• Visualizar histórico de movimentações
• Definir orçamentos mensais
• Criar metas de poupança

Tecnologias
React 19 + Vite + Supabase + Vercel
